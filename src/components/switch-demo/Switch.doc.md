### API

| 参数         |说明        |类型 |默认值 |可选值 |
| ------------ | :----------------|:------- | :----- | :----- |
| value |受控模式下的值 | Boolean |false |true |
| disabled      |设置失效状态  |Boolean |false |true |

### Events

| 事件名称 |      说明      | 类型 |
| -------- | :------------: | -------: |
| on-update:value   | 当 value 改变时触发的回调函数 | (value: Boolean) => void |
