### API

| 参数         |        说明        |     类型 | 默认值 | 可选值 |
| ------------ | :----------------: | -------: | -----: | -----: |
| theme         | 设置按钮的主题 |   String |     default |     default、primary、danger |
| disabled | 按钮失效状态 |   Boolean |   false |  true |
| click        |  点击按钮时的回调  | Function |     —— |     —— |

| -------- | :------------: | -------: |
